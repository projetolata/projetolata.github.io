# lata.github.io

Ter um computador hoje não é luxo, é ferramenta básica de sobrevivência para quem precisa estudar e/ou trabalhar. Mas quando o orçamento é apertado, comprar a primeira máquina se torna um campo minado.

O mercado vende falsas promessas para pessoas de baixa renda. Lojas pegam processadores de baixíssimo custo (como Celerons), instalam pouca memória RAM, um HDD lento, um Windows pesado para as específicações e vendem a preço de ouro como "o PC ideal para estudos/jogos!" O resultado: Um PC que evaporou todo o sagrado e suado dinheiro do trabalhador e, em poucos dias, travará tanto que se tornará lixo eletrônico precocemente.

O projeto **LATA** é uma auditoria de hardware comunitária para combater isso. Nós mapeamos computadores e peças antigas, baratas e guerreiras capazes de entregar desempenho real por um preço minúsculo. Não caia no golpe da vitrine. Salve seu orçamento.
